• svg is a root element.
  • XML is case sensitive.
  • You must close all tags.
  • Attributes must be inside quotes.
  • doctype declaration should be left off - leads to more problems than it solves.
  • To identify the version of the SVG for other types of validation the version and baseProfile attributes should always be used instead.
  • The order of rendering of elements matters - later elements are rendered atop previous elements.
  • file types: .svg & .svgz
